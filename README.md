# SpringBoot整合mybatis
springbootdemo

----------

# 说明 #
使用spring boot+mybatis+thymeleaf+mysql做的玩具

## 技术点 ##
- 整合mybatis
- thymeleaf模板

## 注意事项 ##
- jdk1.8
- spring boot 1.4.1
- 127.0.0.1:8080/demo/findAll进入
- 数据库mysql
